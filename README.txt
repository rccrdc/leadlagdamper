(The code was produced and validated on MATLAB R2017a with Microsoft Windows 10 OS)
1) Open the file "Run_Model.m", run the script.
This script will run the model by integrating the differential equation system and plotting the variables of interest.

ITA
(Il codice è stato prodotto e validato su MATLAB R2017a con OS Microsoft Windows 10)
1) Aprire il file "Run_Model.m", run lo script.
Tale script farà run del modello integrando il sistema di equazioni differenziali e plottando le variabili di interesse.
2) Alla fine, selezionare il file "UnIstalling.p" e premere il tasto "F9", altrimenti selezionare il file, premere il tasto destro e cliccare su "Run".
N.B. : Non modificare la cartella "Code".
